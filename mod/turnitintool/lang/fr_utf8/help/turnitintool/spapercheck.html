Exercices Turnitin - Comparer avec la base de données documentaire des étudiants

Cette option permet à l'enseignant de choisir la base de données Turnitin qui sera comparée aux travaux des étudiants lors de l'élaboration du Rapport de Similitude. Il est possible que le pourcentage de Similarité diminue si une option de stockage est dé-sélectionnée.
