Echo360 LTI Launch Filter
=========================
Echo360 is a filter that allows you to view embedded videos for Moodle assignments.


Installation
=========================
If you are uploading Echo360 LTI Launch Filter, first expand the zip file and upload the echo360 folder into:
[PATH TO MOODLE]/filter.

Then visit your Moodle server's Site Administration page. Moodle will guide you through the installation.

After installing it, you will need to enable the Echo360 filter. You can enable it by visiting:
Site Administration / Plugins / Filters / Manage Filters and then setting to "On" the Echo360 LTI Launch Filter.
